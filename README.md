# Development

## Server

1. Install dependencies `yarn install`
1. Start development server `yarn dev`

### Linting

[Setup Guide Used](https://blog.logrocket.com/how-to-set-up-node-typescript-express/)

```
yarn lint
```

## Client

1. Install dependencies `yarn install`
1. Start development server `yarn dev`
